template<class T, class R>
class BrillouinZoneGrid3 : public InterpolateGrid3<T, R>¶

A class to hold both a BrillouinZone and InterpolateGrid3.

By adding a BrillouinZone object to an InterpolationGrid3 object, it is possible to interpolate at arbitrary points within the first Brillouin zone.

BrillouinZoneGrid3(const BrillouinZone &bz, const size_t *n)¶

Construct using number of steps

Note

The number of steps along each dimension of the InterpolationGrid3 is a function of n. For a given dimension, i, N[i] = 2*n[i]+1 if n[i]>0 or N[i] = 1 if n[i]==0

Parameters

  • bz: The BrillouinZone object

  • n: The three number of steps

void truncate_grid_to_brillouin_zone()¶

Remove unusable grid points.

Utilizing methods of the BrillouinZone object, determine which grid points are inside of the first Brillouin zone or within one-step of such a point and assign their map indexes sequentially. For all points not inside the first Brillouin zone or within one-step of such a point, set the map index to an invalid value.
