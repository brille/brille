class LatVec¶

Superclass to identify both LDVec and LQVec.

The two Lattice vector classes, LDVec and LQVec, are subclasses of ArrayVector. In order to distinguish between the Lattice vector types and “bare” ArrayVector objects in operator- and function-overloading it is advantageous to have a shared superclass on which to enable templates. Thus LatVec is a superclass used for logic only which has no properties and defines no methods.
