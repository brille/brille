bool intersect_at(const LQVec<double> &ni, const LQVec<double> &pi, const LQVec<double> &nj, const LQVec<double> &nk, LQVec<double> &intersect, const int idx)

A specialization where two planes pass through the origin.

Parameters

  • ni: A LQVec<double> reference to the normal for plane i

  • pi: A LQVec<double> reference to the on-plane point for plane i

  • nj: A LQVec<double> reference to the normal for plane j, which passes through the origin

  • nk: A LQVec<double> reference to the normal for plane k, which passes through the origin

  • [out] intersect: A LQVec<double> reference to a list of intersection points

  • idx: The location in intersect where the found intersection point should be stored, if it exists and is not too far from the origin.
