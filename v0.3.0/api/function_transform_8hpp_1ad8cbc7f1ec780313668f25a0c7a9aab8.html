template<class T, typename S = typename std::common_type<T, invPtype>::type>
LDVec<S> transform_to_primitive(const Direct &lat, const LDVec<T> &a)

transform_to_primitive(Direct, LDVec)

Takes a Direct lattice, plus a direct lattice vector expressed in the same standard (non-primitive) lattice and returns an equivalent lattice vector expressed in the primitive lattice. The two lattice vectors have different indices but have the same Å⁻¹ representation.
