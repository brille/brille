template<typename T>
class ArrayVector¶

A class to hold a vector of arrays in contiguous memory.

This class holds N arrays, each with M elements in a N*M contiguous block of memory.

template<class Integer, typename = typename std::enable_if<std::is_integral<Integer>::value>::type>
ArrayVector(const T *d, const std::vector<Integer> &shape, const std::vector<Integer> &strides)¶

ArrayVector contstructor taking shape and stride information for the case of a non-contiguous and/or non-row-ordered array at the provided pointer.

Note

The strides vector must indicate the number of bytes necessary to move from one point to the next along a given dimension of the input data array. The ArrayVector is strictly two dimensional and therefore a convention must be adopted for converting the D-dimensional input array [D=shape.size()]. This constructor will attempt to combine the 2ⁿᵈ through Dᵗʰ dimensions such that (n,m) = (shape[0],shape[1]*…*shape[D-1])

Parameters

  • d: a pointer to the n*m block of data, to be copied

  • shape: a vector of sizes along each dimension of the input array

  • strides: a vector of strides along each dimension of the input array

A new _data block is created even if the size does not change. As much of the old _data block as will fit is copied into the new block.

int removeelements(const size_t from, const size_t to)¶

Truncate the arrays by removing elements

Note

If to is greater than the number of elements in each array, the removal will stop at the array end.

Return

0 if the removal is successful, 1 if from is greater than the number of starting elements, or 2 if from=1 and to>=[number of initial elements]-1

Note

A non-zero return indicates that nothing was done to the _data block, otherwise a copy has been preformed.

Parameters

  • from: the first element which will be removed from each array

  • to: the last element which will be removed from each array
