template<typename T>
class LDVec : public LatVec, public ArrayVector<T>¶

3-vector(s) expressed in units of a Direct lattice

By adding a Direct lattice to a 3-element ArrayVector this class represents one or more 3-vector in units of a real-space-spanning lattice.
