template<typename T>
class LQVec : public LatVec, public ArrayVector<T>¶

3-vector(s) expressed in units of a Reciprocal lattice

By adding a Reciprocal lattice to a 3-element ArrayVector this class represents one or more 3-vector in units of a reciprocal-space-spanning lattice.
