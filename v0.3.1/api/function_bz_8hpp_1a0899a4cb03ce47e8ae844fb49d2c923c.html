bool three_plane_intersection(const LQVec<double> &n, const LQVec<double> &p, const ArrayVector<double> &xyz, const int i, const int j, const int k, LQVec<double> &iat, const int idx = 0)

Find the intersection point of three planes if it exists.

From vectors to points on three planes and their normals determine if an intersection point exists and is not too large.

Parameters

  • n: A pointer to the plane normal vectors

  • p: A pointer to the points on each plane

  • xyz: A pointer to the plane normals in orthonormal units

  • i: index into n, p, and xyz for the first plane

  • j: index into n, p, and xyz for the second plane

  • k: index into n, p, and xyz for the third plane

  • [out] iat: A LQVec to hold the intersection point, if it exists

  • idx: Where the intersection point should be stored in iat
